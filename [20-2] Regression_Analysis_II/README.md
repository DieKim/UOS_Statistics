# 회귀분석II
회귀분석I 과목에서 학습한 회귀분석 기본 이론을 기초로 비선형모형으로 확장된 회귀분석 이론 학습

회귀분석의 이론적인 내용 학습과 더불어 데이터 분석을 학습함으로써 실제 데이터에 회귀분석 이론을 적용하는 능력 수양

<br>

## 💡 Main Topics
- 비선형 회귀모형: PROC NLIN
- 시계열 회귀모형: PROC AUTOREG
- 로지스틱 회귀모형: PROC LOGISTIC
- 일반화 선형모형: PROC GENMOD
